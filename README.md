# Login system

## Apraksts
	Pieteikšanās sistēma, kurā var reģistrēties. Tas rakstīs un nolasīs datumu no failiem c ++ 
## Galvenās funkcijas
	- Reģistrēties
	- Pieteikties
	- Izmantojiet datus no faila
  	- Salīdziniet, vai dati ir ievadīti pareizi
## Izmantotās tehnoloģijas
	- C++
	- fstream, iostream
